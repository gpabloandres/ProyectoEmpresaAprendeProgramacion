Introducción a la Programación

La programación es el arte de dar instrucciones a una computadora para que realice tareas específicas. A este conjunto de instrucciones se lo denomina algoritmo

Ejemplo de un algoritmo para hacer un té:

                
                    Calentar agua
                    Colocar una bolsita de té en una taza
                    Verter agua caliente en la taza
                    Esperar unos minutos
                    Retirar la bolsita de té

Ejemplo de la programación (usando el lenguaje de programación python) del algoritmo para hacer un té:

                
                    # Algoritmo para hacer una taza de té en Python
                    calentar_agua()
                    colocar_taza()
                    verter_agua_caliente()
                    esperar_minutos(3)
                    retirar_bolsita()

Tipos, Variables y Operadores

Los tipos de datos más comunes son:

  • Enteros (int)
  • Decimales (float)
  • Lógicos (bool)
  • Cadenas (str)

Las variables son contenedores que almacenan a esos datos. Por ejemplo en Python los datos se almacenan en variables de la siguiente manera:

                
                    # Declaración y definición de las variables
                    edad = 25
                    promedio = 92.5
                    nombre = "Ana"
                    
                    # Mostrar variables
                    print("Edad:", edad)
                    print("Promedio:", promedio)
                    print("Nombre:", nombre)

Las operadores sirven para operar con esos datos. Existen diferentes tipos de operadores:

  • Aritméticos: Para hacer cuentas con tus datos y devolver un número (Ejemplo: sumar, restar, multiplicar, dividir, etc...)
  • Relacionales: Para comparar tus datos y devolver VERDADERO O FALSO (Ejemplo: Si el numero1 > numero2 devuelve VERDADERO)
  • Lógicos:
Fijate los siguiente ejemplos de uso de operadores: 
                
                    # Uso de operadores aritméticos para obtener un resultado numérico
                    print("Suma:", 3 + 4)  
                    print("Resta:", 3 - 2)
                    print("Multiplicación:", 8 * 9)
                    print("División:", 16 / 2)
                    print("Potenciación:", 2 ** 3)

                    # Uso de operadores relacionales para obtener como resultado VERDADER o FALSO
                    print("Mayor que:", 1 > 2)  
                    print("Menor que:", 1 < 2)
                    print("Igual a:", 2 == 2)

Estructuras de control de flujo

Sirven para ordenar y decidir cómo y de qué manera se ejecutarán las líneas de código. Existen diferentes estructuras:

  • Condicionales (if-else-elif): Las líneas de código dentro de este tipo de estructura, se ejecutan sólo sí se cumple una condición
  • Bucles (while-for): Las líneas de código dentro de este tipo de estructura, se repiten hasta que no se cumpla una condición

Ejemplo en Python del uso de la estructura de control de flujo Condicional:

                
                    # Ejemplo del uso del Condicional if-else-elif
                    nota = 85
                    if nota == 10:
                        print("¡Aprobado con Excelente!")
                    elif nota >= 60 and nota <=10:
                        print("¡Aprobado!")
                    else:
                        print("¡Reprobado!")

Ejemplo en Python del uso de la estructura de control de flujo Bucle:

                
                    # Ejemplo del uso del Bucle for
                    for i in range(5):
                        print("Número:", i)
                    
                    # Ejemplo del uso del Bucle while
                    contador = 0
                    while contador < 3:
                        print("Contador:", contador)
                        contador += 1
